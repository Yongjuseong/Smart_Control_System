# Smart_Control_System
IoT_Smart_Control_System / By using Python 3.8 and Raspberry Pi

Control Architecture.
R -> LED Red On
G -> LED Green On
B -> LED Blue On
P -> Start PIR sensor , If it is, it will turn on LED Red light. If it is again, then it will be blink.
M -> LED's Brightness adjustment
X -> LED Blink
Button Press -> Buzzer On, If you press it again, all will be off.
S -> Sound On ( by using buzzer and you can choose 5 type's of musics.)
O -> All Off
